# Wildharvest licensing status

No open-source license has been selected for Wildharvest.

Until the project owner chooses and adds a license, the source code, artwork, documentation, and release archives remain copyright-protected and are not offered under an open-source license. The absence of a license does not grant permission to copy, modify, redistribute, or publish the project.

`module.json` intentionally omits the `license` field. This is more accurate than naming a license whose text and terms have not been accepted by the owner.

Before public distribution, the project owner must:

1. choose the intended license and confirm that it covers every distributable project-owned asset;
2. add the complete license text as `LICENSE`;
3. set `module.json.license` to `LICENSE` or to the authoritative license URL;
4. review third-party dependencies, Foundry VTT, the D&D5e system, compendium content, fonts, icons, and other externally sourced materials under their own terms.

Local private development releases may continue without making a public license claim. This document records project status; it is not itself a software license.
