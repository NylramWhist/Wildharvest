# Retired content project

The former companion module was permanently retired before Wildharvest 0.90.0.

Removed from the active workspace:

- the complete `gathering-content/` module source and compiled packs,
- its authoring sources, templates, and LevelDB builder,
- `tools/create-content-module-backup.ps1`,
- installation recommendations and descriptions that treated it as an optional companion,
- the redundant `backups/gathering-content-v0.1.0` directory and ZIP.

One final local safety archive is retained outside Git:

- `backups/retired-gathering-content-0.2.0-stage-09-pre-removal.zip`

The main module no longer bundles, recommends, requires, builds, validates, or backs up companion content. A GM must select Item compendiums supplied by the world, the game system, or another independently managed source.

## No compatibility layer

Wildharvest is a clean module and contains no migration for the retired project. Historical names remain only in this retirement record, the audit, old changelog entries, and local backup filenames. They are not runtime dependencies and must not be restored to active development.
